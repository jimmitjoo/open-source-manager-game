# Open Source Manager Game

This is an open source project, that is created for developers to create the manager game of the future.

## Official Documentation

There are no official documentation just yet, but it will be in the future.

## Contributing

Thank you for considering contributing to the manager game! We does not have a guide yet, but we build this game upon the Laravel framework, which has its documentation [here](http://laravel.com/docs/contributions).

## Security Vulnerabilities

If you discover a security vulnerability within the game, please send an e-mail to Jimmie Johansson at jimmitjoo@gmail.com. All security vulnerabilities will be promptly addressed.
